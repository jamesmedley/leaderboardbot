Leaderboard Bot

Documentation

Usage

All bot commands must begin with the prefix "!lb".

Leaderboard

To request the top ten leaderboards, use the base command "!lb".

To switch between leaderboards for different awards, use the select menu beneath the embed.

Streaks

To see who currently holds streaks, use "!lb s".

User Performance

To view the performance data of a particular user, use "!lb @user".

Manual Awarding

To manually award a win to a user, use the command "!lb wu/lm @user", where "wu" indicates the Waking Up Early Award and "lm" indicates the Last Message Of The Day Award.
